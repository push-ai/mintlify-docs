  **Prerequisite** You should have admin access to one of our supported data
  sources. If you'd like help with navigating setup, you can [reach out to

<Steps>
  <Step title="Connect a data source and chat workspace">
    Connect with one of our supported data sources, using either a semantic
    layer or data warehouse integration.
  </Step>
  <Step title="Define metrics, dimensions and metadata">
    Define metrics and dimensions within Push.ai or connect with a semantic
    layer.
  </Step>
  <Step title="Use Reports and Subscriptions to receive updates">
    Use Subscriptions to receive a personalized feed of updates, including AI
    Summaries. Use Reports to send updates to shared company channels.
  </Step>
  <Step title="Invite your team">
    Invite Business Users to set up their own Subscriptions and access data
    inside Push.ai.
  </Step>
</Steps>

## Troubleshooting

Here's how to solve some common problems when setting up Push.ai.

<AccordionGroup>
  <Accordion title="I can't connect to my data source">
    Update to Node v18. Run `mintlify install` and try again.
  </Accordion>
  <Accordion title="My data source schema is not loading">
Go to the `C:/Users/Username/.mintlify/` directory and remove the `mint`
folder. Then Open the Git Bash in this location and run `git clone
https://github.com/mintlify/mint.git`.

Repeat step 3.

  </Accordion>
  <Accordion title="Getting an unknown error">
    Try navigating to the root of your device and delete the ~/.mintlify folder.
    Then run `mintlify dev` again.
  </Accordion>
</AccordionGroup>

Need additional help? Feel free to [contact us.](mailto:support@push.ai)
